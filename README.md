Because I always forgot to set up something, I just wanna clone this and be ready.

- git
- bash
- libnotify
- plan9ports
- golang
- go fonts
- cmu fonts

- xterm

- chromium
- feh (background set up)
- xconsole
- xbindkeys (hotkey set up)
- fontconfig
- stalonetray (tray icon window)
- dunst (freedesktop notifications)
